== Description == 

Insire button code for donations or payments via MOIP www.moip.com.br in pagina 

Em pt_BR: Insere botao shortcodo [doarmoip] para doar com MOIP

Admin WP/ferrammentas/Donations/register of donations

Se ja tem conta moip podera gerar o codigo de botao de doar em 

https://www.moip.com.br/admdonation.do?method=viewdonations ou Crie sua conta em

https://www.moip.com.br/CreateWallet.do

Comentarios: Criei este plugin devido a erros ao tentar insirir codigo html de botao de doar moip 

diretamente em post ou paginas seja atraves do Windows Live Writer ou editor online wordpress

URL readme: http://www.andrewebmaster.com.br/internet/wp-content/plugins/doarmoip 

URL readme: http://www.andrewebmaster.com.br/internet/wp-content/plugins/doar/readme.txt 

doarmoip allows Wordpress users to embed a ShortCode Moip Donation button. ([doarmoip]) 

Agora apoio de Idiomas Portugues Brasil pt_BR.PO, Ingles USA us_US.PO, Espanhol Espanha es_ES.PO

== Installation ==

1.First, download the plugin, upload it to your plugin directory and activate it on WordPress
  
http://wordpress.org/extend/plugins/doarmoip ative plugin

2.Go to www.moip.com.br in tools/Donation/Code

3.In WordPress go to manage then the Tools submenu/donate MoiP 

4.Paste the code in the box and save to insert the donation button with system in pages or post, a simple brand is everything you need!

Insert [doarmoip] anywhere in their contegorias or Post and pages WordPress

== Descricao ==

em Portugues pt-BR:
doarmoip permite que usuarios de WordPress incorporar um botao de doar do 

ShortCode MoIP  [doarmoip]

Comentarios:

Criei este plugin devido a erros ao tentar insirir codigo html de botao de doar moip 

diretamente em post ou paginas seja atraves do Windows Live Writer ou editor online wordpress

Insire  shortcode para botao doar com ou pagamentos  via MOIP

www.moip.com.br pagina ferrammentas/doacoes/registo para doar

https://www.moip.com.br/admdonation.do?method=viewdonations 


== instalacao == 

1-Download  plugin em: http://wordpress.org/extend/plugins/doarmoip/ Upload  da pasta dooarmoip para o diretorio /wp-content/plugins , Ative o plugin.


2-No Wordpress va para Gerenciar, em sua admin wordpress em seguida, em/ferramentas do sub-menu 
/doar Moip,Voce poder ver isso em imagem screenshot-5 ou em imagens de tutorial em: http://www.andrewebmaster.com.br/internet/?p=1046 


3-Cole o codigo  de doar gerado em sua area de conta MOIP na caixa e salve para inserir o botao de doar com o

sistema em paginas ou post, uma marca simples apenas isso! 

4-Digite [doarmoip] em qualquer lugar no seu Categorias Post ou paginas
